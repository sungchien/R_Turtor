2.2 R基本資料型態 | integer & numeric

  • 數值資料
    • integer：整數
    • numeric：實數

2.3 R基本資料型態 | integer & numeric

  • 數值運算
    • + 加
    • - 減
    • * 乘
    • / 除
    • ^ 次方
    • %% 整數除法的餘數
    • %/% 整數除法的商

2.4 練習

  • 在Console下輸入下面的敘述，分別查看c, d, e
a <- 5
b <- 3
c <- 5/3
d <- 5%/%3
e <- 5%%3

2.5 R基本資料型態 | integer & numeric

  • 數值函數
    • abs() 絕對值
    • sqrt() 平方根
    • round() 四捨五入

2.6 練習

  • 在Console下輸入下面的敘述，分別查看f, g
f <- sqrt(2^4)
g <- round(c)

2.7 R基本資料型態 | character

  • character資料表示
  • 在Console上輸入下面指定敘述，並且查看strR和strVar
strR <- "R"
strVar <- "Langague"

2.8 R基本資料型態 | character

  • character資料的函數
    • paste() 字串連接
    • substr() 取出字串的一部份

2.9 練習

  • 在Console上輸入下面敘述，並且查看運算結果
strCon <- paste(strR, strVar)
strSub <- substr(strVar, 1, 4)

2.10 R基本資料型態 | logic

  • TRUE(真)與FALSE(偽)
  • 邏輯運算
    • | 或
    • & 且
    • ! 否

2.11 練習

  • 在Console上輸入下面敘述，並且查看運算結果
logValue1 <- 3>5
logValue2 <- "R Language" == paste(strR, strVar)
logValue3 <- logValue1 | logValue2
logValue4 <- logValue1 & logValue2
